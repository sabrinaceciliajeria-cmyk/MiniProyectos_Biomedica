# Glosario de comandos y palabras clave usadas en Mini Proyecto 1

| Comando / palabra | Explicación |
|------------------|-------------|
| `def`            | Define una función en Python, para agrupar código que realiza una tarea específica. |
| `return`         | Devuelve un valor desde una función al lugar donde fue llamada. |
| `if`             | Condicional que ejecuta un bloque de código solo si la condición es verdadera. |
| `elif`           | “Else if”: se ejecuta si las condiciones anteriores no se cumplen y esta sí. |
| `else`           | Se ejecuta si ninguna condición anterior se cumple. |
| `while`          | Ciclo que se repite mientras la condición sea verdadera. |
| `break`          | Termina inmediatamente un ciclo (`while` o `for`). |
| `input()`        | Solicita al usuario que ingrese datos desde teclado. |
| `float()`        | Convierte un valor a número decimal. |
| `list.append()`  | Agrega un elemento al final de una lista. |
| `round(valor, n)`| Redondea un número a `n` decimales. |
| `f"{var}"`       | F-strings: permite insertar variables dentro de texto formateado. |
| `nombre.lower()` | Convierte un texto a minúsculas, útil para comparar sin importar mayúsculas. |
| `for ... in ...` | Ciclo que recorre todos los elementos de una lista o secuencia. |
| `<n` en f-strings| Alinea el texto a la izquierda en un ancho de `n` caracteres, útil para crear tablas. |
